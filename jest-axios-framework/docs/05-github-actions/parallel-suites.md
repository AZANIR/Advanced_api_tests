# Паралельні тестові сюїти

## Опис

Паралельний запуск різних тестових сюїтів для швидшого виконання.

## Файл

`.github/workflows/parallel.yml`

## Паралельні джоби

1. **Smoke Tests** - Швидкі базові тести
2. **Functional Tests** - Функціональні тести
3. **Negative Tests** - Негативні тести
4. **Integration Tests** - Інтеграційні тести

## Переваги

- ✅ Швидше виконання (паралельно)
- ✅ Незалежність джобів
- ✅ Детальні звіти для кожного сюїту

## Summary Job

Після завершення всіх джобів запускається summary job, який:
- Очікує завершення всіх тестових джобів
- Створює summary з результатами
- Працює навіть якщо деякі джоби упали (`if: always()`)

## Додавання нових сюїтів

Додайте новий джоб для додаткового тестового сюїту:

```yaml
new-suite:
  name: New Suite Tests
  runs-on: ubuntu-latest
  steps:
    # ... кроки
```
