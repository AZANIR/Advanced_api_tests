# Scheduled Runs (Cron Jobs)

## Опис

Автоматичний запуск тестів за розкладом використовуючи cron.

## Файл

`.github/workflows/scheduled.yml`

## Cron налаштування

```yaml
schedule:
  - cron: '0 2 * * *'  # Щодня о 2:00 UTC
```

### Формат cron

```
┌───────────── хвилина (0 - 59)
│ ┌─────────── година (0 - 23)
│ │ ┌───────── день місяця (1 - 31)
│ │ │ ┌─────── місяць (1 - 12)
│ │ │ │ ┌───── день тижня (0 - 6, 0 = неділя)
│ │ │ │ │
* * * * *
```

### Приклади

- `0 2 * * *` - Щодня о 2:00 UTC
- `0 */6 * * *` - Кожні 6 годин
- `0 0 * * 1` - Кожен понеділок о 00:00
- `0 0 1 * *` - Першого числа кожного місяця

## Manual Dispatch

Workflow також можна запустити вручну:

```yaml
workflow_dispatch:
```

## Збереження артефактів

Артефакти зберігаються на 30 днів:

```yaml
retention-days: 30
```

## Summary

Workflow створює summary з інформацією про запуск:
- Run Number
- Date
- Status
